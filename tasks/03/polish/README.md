# Вычисление обратной польской нотации

Реализуйте функцию `Calculate(expr string) (int, error)`, 
которая вычисляет значение выражения, записанного в [обратной польской нотации](https://en.wikipedia.org/wiki/Reverse_Polish_notation).

На вход функция получает строку из **токенов**, разделённых произвольным количеством пробельных символов ([`strings.Fields`](https://pkg.go.dev/strings#Fields)): целые литералы (в том числе с знаком, например `-12`, и многозначные), операторы `+`, `-`, `*`, `/`.

- Бинарные операторы (`+`, `-`, `*`, `/`) применяются к двум верхним числам на стеке: для `-` и `/` это «предпоследнее» минус/делить «последнее» (как в классической RPN: `12 15 -` даёт `-3`, `17 5 /` даёт `3`). Целочисленное деление — как в Go (`/` для `int`).
- Символ `-` **унарный**, если на стеке ровно одно число: оно заменяется на противоположное по знаку.
- Деление на ноль — невалидное выражение (`ErrInvalidExpression`).

Помимо значения выражения функция возвращает ошибку в случае, если само выражение оказалось невалидно.
В таком случае вам нужно вернуть уже существующую ошибку `ErrInvalidExpression`. 
Примеры невалидных выражений вы можете найти в тестах.

Пример:
```go
res, err := Calculate("3 5 +")
fmt.Println(res, err)  // 8 nil
```

Вам могут пригодиться:
- [strings.Fields](https://pkg.go.dev/strings#Fields)
- [strconv.Atoi](https://pkg.go.dev/strconv#Atoi)
