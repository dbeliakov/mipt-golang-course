# ROT13 для hex-строки

Напишите функцию `Rot13`, которая принимает hex-закодированную строку, декодирует её, применяет шифр ROT13 **на месте** (in-place) к латинским буквам (a-z, A-Z) в декодированном буфере и возвращает результат в виде hex-закодированной строки. Остальные символы не изменяются. При ошибке декодирования hex функция должна вернуть ошибку.

ROT13 сдвигает каждую букву на 13 позиций в алфавите (циклически): a→n, b→o, … z→m.

Вместо типа char в Go используется тип `rune`. Над `rune` можно производить операции с другими `rune` и числами:
```go
var r rune
r += 'a'
r += rune(13)
```
