# Reverse Each Word

Реализуйте функцию `ReverseEachWord`, которая принимает строку, состоящую из слов, разделённых одиночными пробелами, и возвращает строку, в которой символы каждого слова записаны в обратном порядке, а порядок слов остаётся прежним.

Например: `"Hello World"` → `"olleH dlroW"`.

Строки могут содержать символы любых языков, а значит представляют собой `UTF-8`-последовательности. Ваша функция должна корректно обрабатывать `UTF-8`-руны, не обрезая их. С этим вам может помочь пакет [`unicode/utf8`](https://pkg.go.dev/unicode/utf8), изучите его возможности.

При обработке слов важно не делать лишних аллокаций. Посмотреть количество аллокаций и запустить бенчмарк можно аналогично задаче [`rle`](../rle/README.md#запуск-бенчмарков-и-анализ-профилей).
